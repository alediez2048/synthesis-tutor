# ENG-064 Primer: Interleave Problem Types Within Sessions

**For:** New Cursor Agent session
**Project:** Fraction Quest — Interactive AI-Powered Fractions Tutor for Ages 8–12
**Phase:** Phase 9: EdTech Research-Informed Redesign
**Date:** Mar 16, 2026
**Research:** Bjork & Bjork, 2011 — interleaving improves long-term retention (20% → 63%). Blocked practice yields higher in-session scores but worse transfer. In-session scores may dip; long-term retention jumps.

---

## What Is This Ticket?

Mix equivalence, comparison, and simplification problems within sessions instead of blocking by type. Currently guided practice and assessment may sequence by concept (GP-1 split, GP-2 build, GP-3 compare, GP-4 simplify). Interleave: alternate problem types so the student must retrieve the right strategy each time. Accept that in-session performance may dip; prioritize long-term retention.
