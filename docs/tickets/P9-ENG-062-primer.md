# ENG-062 Primer: Off-Ramps for Real-World Transfer

**For:** New Cursor Agent session
**Project:** Fraction Quest — Interactive AI-Powered Fractions Tutor for Ages 8–12
**Phase:** Phase 9: EdTech Research-Informed Redesign
**Date:** Mar 16, 2026
**Research:** Hirsh-Pasek et al., 2015 — "Meaningful" pillar: learning must connect to lived experience. Transfer to the physical world is critical.

---

## What Is This Ticket?

When a student shows mastery, the tutor bridges to the physical world instead of pointing deeper into the app. Add off-ramps: "Try splitting a pizza at dinner." "Share a snack equally with a sibling and name the fraction." "Look for fractions when you measure something." This fixes the weakest Hirsh-Pasek pillar — making learning meaningful through real-world application.
