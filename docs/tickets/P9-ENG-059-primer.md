# ENG-059 Primer: Scaffolding Mixing Board (Multi-Dimensional)

**For:** New Cursor Agent session
**Project:** Fraction Quest — Interactive AI-Powered Fractions Tutor for Ages 8–12
**Phase:** Phase 9: EdTech Research-Informed Redesign
**Date:** Mar 16, 2026
**Research:** Kalyuga et al., 2000; Wood, Bruner & Ross, 1976 — scaffolding should be multi-dimensional, not a single difficulty dial.

---

## What Is This Ticket?

Build a scaffolding mixing board that independently controls:
1. **Problem difficulty** — easier vs. harder fraction problems
2. **Visual support level** — CRA progression (Concrete → Representational → Abstract): blocks vs. diagrams vs. symbols
3. **Guidance level** — how much the tutor prompts, hints, or demonstrates

A kid can be ready for harder problems while still needing the blocks, or vice versa. The current LangGraph/state machine adjusts difficulty as one dimension; this ticket expands to three independent axes.
