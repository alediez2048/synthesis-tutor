# Number Line Model: Fractions as Distances and Positions

The number line is the most mathematically powerful fraction model and the one most aligned with how fractions function in later mathematics. CCSS (2010) standard 3.NF.2 explicitly requires students to represent fractions on a number line. Yet it is also the most cognitively demanding model for young learners.

## Why the Number Line Matters

Unlike area models, the number line treats fractions as *numbers* — not just parts of a shape. This is critical for ordering (which fraction is larger?) and for understanding that fractions exist between whole numbers. Siegler et al. (2011) found that number-line estimation accuracy in elementary school is one of the strongest predictors of later math achievement, including algebra.

## Teaching Ordering

Place 0 and 1 on a line. Ask the student to place 1/2, then 1/4, then 3/4. The spatial reasoning required — "1/4 is closer to 0 than 1/2" — builds magnitude sense that area models do not develop as effectively. Extend to comparing fractions with unlike denominators: "Where does 2/3 fall relative to 3/4?"

## Teaching Addition

Number-line addition is modeled as concatenating jumps. Show 1/4 + 1/4 as two equal hops landing on 2/4 (= 1/2). For unlike denominators, demonstrate how the jump sizes must first be converted to a common unit. This reinforces that common denominators are not an arbitrary rule but a requirement for combining distances of different sizes.

## Teaching Equivalence

Two fractions are equivalent if they land on the same point. This definition is simple and precise. Have students partition the line into different denominators and discover shared positions.

## Tutor Guidance

Introduce the number line after students have built intuition with area models. If a student struggles with ordering, the number line is the go-to intervention. Van de Walle et al. (2018) recommend using it alongside, not instead of, area models to build representational fluency.
