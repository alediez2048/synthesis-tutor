# Using Student Errors as Teaching Moments

Errors in fraction work are not random — they reveal systematic reasoning patterns that, once understood, become powerful levers for instruction. Ashlock (2010) catalogued common fraction error patterns and found that most stem from overgeneralizing whole-number rules to fractions.

## Common Error Patterns

**Adding denominators** (same_denominator): 1/3 + 1/4 = 2/7. The student treats fraction addition like whole-number addition of each component. Address by asking: "If you ate 1/3 of a pizza and then 1/4 of the same pizza, did you eat 2/7 of it? Does that make sense?" **Using whole-number comparison** (used_whole_number): believing 1/3 > 1/2 because 3 > 2. Correct with a visual model — show both fractions on the same number line. **Flipped fraction** (flipped_fraction): confusing numerator and denominator roles. Reinforce with consistent language: "the bottom number tells how many equal parts; the top tells how many you have."

## Growth Mindset Framing

Dweck (2006) demonstrated that students who view mistakes as learning opportunities outperform those who view them as indicators of low ability. When a student makes an error, respond with curiosity, not correction: "Interesting — tell me how you got that." Then guide them to discover the error themselves. Boaler (2016) found that reframing mistakes as "brain growth" significantly improved math attitudes and performance.

## Error Analysis as Assessment

Errors tell you more than correct answers. A student who writes 2/4 = 1/4 (off_by_one in simplifying) understands that simplifying is needed but has a procedural gap. A student who writes 1/2 + 1/3 = 1/5 has a deeper conceptual gap. Tailor your response to the error type.

## Tutor Guidance

Never skip past errors. Each error is diagnostic data. Acknowledge the student's reasoning ("I can see why you might think that"), identify the specific misconception, and address it with the most concrete representation available. Celebrate the correction: "You just fixed your own thinking — that's real learning."
